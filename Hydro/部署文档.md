# 编译未来 OJ 平台 - Debian 12 完整部署指南

> 基于 Hydro OJ 定制的信息学奥林匹克竞赛培训平台

## 🎯 平台特色

- **专业教学环境**：GoC 图形化编程 + C++ 竞赛编程
- **完整题库**：内置一本通编程启蒙题库 480+ 题目
- **智能 IDE**：在线代码编辑器，支持多语言实时错误检测
- **评测系统**：基于 Go-Judge 的高性能在线评测

## 📋 系统要求

### 硬件配置
- **CPU**: 最低 2 核，推荐 4 核
- **内存**: 最低 4GB，推荐 8GB
- **硬盘**: 60GB 可用空间
- **网络**: 公网 IP（可选域名绑定）

### 系统环境
- **操作系统**: Debian 12 (推荐) / Ubuntu 22.04
- **权限**: 需要 root 用户权限
- **端口**: 8888(Web), 5050(沙箱), 27017(MongoDB)

## 🚀 一键部署（推荐）

### 1. 克隆项目
```bash
# 克隆项目到服务器
git clone https://github.com/DLshujin/bywl_OJ.git
cd bywl_OJ

# 给部署脚本执行权限
chmod +x deploy.sh
```

### 2. 执行一键部署
```bash

./deploy.sh

# 等待部署完成（约 15-30 分钟）
```

### 3. 验证部署
```bash

pm2 status

# 访问网站
curl http://localhost:8888
```

**部署完成！** 访问 `http://服务器IP:8888` 开始使用。

## 🔧 手动部署（详细步骤）

### 第一步：环境准备

#### 1.1 更新系统
```bash
apt update && apt upgrade -y
```

#### 1.2 安装基础工具
```bash
apt install -y curl wget git build-essential python3 python3-pip \
    software-properties-common gnupg2 apt-transport-https ca-certificates \
    lsb-release unzip vim htop
```

#### 1.3 安装 Node.js 22
```bash
# 添加 NodeSource 仓库
curl -fsSL https://deb.nodesource.com/setup_22.x | bash -
apt install -y nodejs

# 安装 Yarn
npm install -g yarn pm2

# 验证安装
node --version  # 应显示 v22.x.x
yarn --version  # 应显示 1.x.x 或更高
```

### 第二步：数据库安装

#### 2.1 安装 MongoDB 7.0
```bash
# 导入 MongoDB 公钥
curl -fsSL https://www.mongodb.org/static/pgp/server-7.0.asc | \
    gpg -o /usr/share/keyrings/mongodb-server-7.0.gpg --dearmor

# 添加 MongoDB 源
echo "deb [ arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] \
https://repo.mongodb.org/apt/debian bookworm/mongodb-org/7.0 main" | \
    tee /etc/apt/sources.list.d/mongodb-org-7.0.list

# 安装 MongoDB
apt update && apt install -y mongodb-org

# 启动并设置开机自启
systemctl start mongod
systemctl enable mongod
systemctl status mongod
```

#### 2.2 安装 Redis（可选，提升性能）
```bash
apt install -y redis-server
systemctl start redis-server
systemctl enable redis-server
```

### 第三步：安装评测环境

#### 3.1 下载 Go-Judge 沙箱
```bash
# 下载最新版本的 go-judge
wget https://github.com/criyle/go-judge/releases/latest/download/go-judge_1.8.8_linux_amd64 \
    -O /usr/bin/sandbox

# 设置执行权限
chmod +x /usr/bin/sandbox

# 验证安装
/usr/bin/sandbox --version
```

#### 3.2 配置沙箱环境
```bash
# 创建配置目录
mkdir -p ~/.hydro

# 复制沙箱配置文件（项目自带）
cp install/docker/judge/mount.yaml ~/.hydro/

# 创建沙箱启动脚本
cat > ~/.hydro/start_sandbox.sh << 'EOF'
#!/bin/bash
cd /root/.hydro
exec /usr/bin/sandbox -mount-conf mount.yaml
EOF

chmod +x ~/.hydro/start_sandbox.sh
```

### 第四步：部署 Hydro 平台

#### 4.1 安装项目依赖
```bash
# 安装依赖（会自动运行 prepare 脚本）
yarn install

# 构建 TypeScript
yarn build

# 构建前端 UI
yarn build:ui:production
```

#### 4.2 配置数据库连接
```bash
mkdir -p ~/.hydro
cat > ~/.hydro/config.json << 'EOF'
{
  "host": "127.0.0.1",
  "port": 27017,
  "name": "hydro",
  "username": "",
  "password": ""
}
EOF
```

#### 4.3 创建系统用户
```bash
# 创建 judge 系统用户
useradd -m -s /bin/bash judge
echo "judge:judge123" | chpasswd

# 创建用户配置目录
su - judge -c "mkdir -p ~/.hydro"
```

#### 4.4 配置评测服务
```bash
cat > ~/.hydro/judge.yaml << 'EOF'
hosts:
  localhost:
    type: hydro
    server_url: http://127.0.0.1:8888
    uname: judge
    password: judge123
    detail: full
sandbox_host: http://127.0.0.1:5050
EOF

# 复制配置到 judge 用户
cp ~/.hydro/judge.yaml /home/judge/.hydro/
chown judge:judge /home/judge/.hydro/judge.yaml
```

### 第五步：启动服务

#### 5.1 启动核心服务
```bash
# 启动 Hydro 主服务
pm2 start packages/hydrooj/bin/hydrooj.js --name hydro-main

# 启动沙箱服务
pm2 start ~/.hydro/start_sandbox.sh --name hydro-sandbox

# 启动评测服务
pm2 start packages/hydrojudge/bin/hydrojudge.js --name hydro-judge

# 保存 PM2 配置
pm2 save
pm2 startup
```

#### 5.2 创建数据库用户
```bash
# 进入 MongoDB 控制台
node packages/hydrooj/bin/hydrooj.js db

# 执行以下命令创建 judge 用户
```

在 MongoDB 控制台中执行：
```javascript
// 创建 judge 用户
db.user.insertOne({
  _id: 3,
  mail: 'judge@hydro.local',
  mailLower: 'judge@hydro.local',
  uname: 'judge',
  unameLower: 'judge',
  hash: "judge123",
  salt: "",
  hashType: 'sha256',
  regat: new Date(),
  ip: ['127.0.0.1'],
  loginat: new Date(),
  loginip: '127.0.0.1',
  priv: 1,
  avatar: 'gravatar:judge@hydro.local'
})

// 退出
exit
```

#### 5.3 重启服务应用配置
```bash
pm2 restart all
```

### 第六步：导入题库

#### 6.1 安装题库导入插件
```bash
# 安装 hydroac-client 插件
node packages/hydrooj/bin/hydrooj.js install https://hydro.ac/hydroac-client.zip

# 手动添加插件
node packages/hydrooj/bin/hydrooj.js addon add '/root/.hydro/addons/hydroac-client'

# 重启主服务
pm2 restart hydro-main
```

#### 6.2 通过 Web 界面导入题库
1. 访问 `http://服务器IP:8888`
2. 注册第一个用户（自动获得管理员权限）
3. 进入"管理" → "题目管理"
4. 点击"导入题库" → "一本通编程启蒙"
5. 等待导入完成（约 10-15 分钟）

## ⚠️ 常见问题

### Q1: Judge 进程频繁重启
**解决方案**:
```bash
# 停止评测服务
pm2 stop hydro-judge

# 清除频率限制
node packages/hydrooj/bin/hydrooj.js db
db.opcount.deleteMany({})
db.session.deleteMany({})
exit

# 重新启动
pm2 restart hydro-main
pm2 start hydro-judge
```

### Q2: 网站无法访问
**检查步骤**:
```bash
# 检查防火墙
ufw status
ufw allow 8888

# 检查端口占用
netstat -tulpn | grep 8888

# 检查服务日志
pm2 logs hydro-main
```

### Q3: 题库导入失败
**解决方案**:
```bash
# 手动下载题库
wget https://hydro.ac/ybtbas.zip -O /root/ybtbas.zip
unzip /root/ybtbas.zip

# 通过管理界面本地导入
```
